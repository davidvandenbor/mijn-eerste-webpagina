Deze paragraaf ziet er visueel heel anders uit dan de andere paragrafen die eronder staan. Dat komt omdat je met CSS speciale selectoren kan aanmaken die "class" heten. Deze kun je vervolgens aan HTML elementen koppelen om ze uniek vorm te geven. Kijk maar naar het HTML attribuut die begint met "class", in de code van je HTML pagina bij deze paragraaf.
